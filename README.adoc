= Ansible for Red Hat Enterprise Linux Workshop

A Showroom-based workshop covering Ansible automation for Red Hat Enterprise Linux, built on Ansible Automation Platform 2.6.

== Overview

This workshop is available in two formats:

* *Full version* (~3 hours) — 15 modules covering command-line Ansible and Ansible Automation Platform
* *90-minute version* — 7 modules focused on core AAP workflows

== Structure

* `content/` — AsciiDoc workshop modules and navigation files
** `nav.adoc` — Full workshop navigation
** `nav-90.adoc` — 90-minute workshop navigation
* `config/` — Lab instance definitions
* `setup-automation/` — Lab provisioning playbooks
* `runtime-automation/` — Runtime automation playbooks
* `ui-config.yml` — Showroom UI and tab configuration
